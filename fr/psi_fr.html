Guide d&aps;Installation pour le pack en langue française

General Notes

Téléchargez le pack langue en cliquant sur le lien "Psi translation - Français" en haut de cette page.
Placez le fichier dans le répertoire approprié selon votre système d&aps;exploitation, comme décrit ci-dessous :
La langue est réglée automatiquement selon les paramètres de votre système. Si vous voulez la modifier à la main dans Psi, vous pouvez le faire dans les "Profils".

GNU/Linux

Installation comme Utilisateur

Copiez psi_fr.qm dans $HOME/.psi/

Mac OS X

Installation pour tout le système

Copiez psi_fr.qm dans Applications/Psi.app/Contents/Resources/
Pour cela faîtes un clic-droit sur Psi dans le répertoire des applications, puis choisissez "Afficher le contenu du paquet". Allez dans le dossier "Contents" puis "Resources". Placez votre psi_fr.qm ici.

Windows 2000/XP/Vista

Installation comme Utilisateur

Copiez psi_fr.qm dans %USERPROFILE%\psidata.

Installation avec les droits Administrateur

Copiez psi_fr.qm dans C:\Program Files\Psi. (Si vous avez installé Psi dans un autre répertoire modifiez le chemin en conséquence.)
