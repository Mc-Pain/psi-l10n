Installationsanleitung für das deutsche Sprachpaket

Allgemeine Hinweise

Laden Sie das Sprachpaket durch einen Klick auf den Link "Psi translation - Deutsch" herunter.
Platzieren Sie die Datei in den für Ihr Betriebssystem angebrachten Ordner, so wie es unten beschrieben ist.
Die Sprache wird automatisch anhand der Systemeinstellungen gewählt. Wenn Sie die Sprache in Psi manuell ändern möchten, wählen Sie "Profil wechseln" aus dem Psi-Menü.

GNU/Linux

Installation als Benutzer

Kopieren Sie psi_de.qm nach $HOME/.psi/.

Mac OS X

Systemweite Installation

Kopieren Sie psi_de.qm nach Programme/Psi.app/Contents/Resources/.
Klicken Sie dazu mit der rechten Maustaste auf Psi im Ordner "Programme" und dann auf "Paketinhalt anzeigen". Im Verzeichnis "Contents" ist das Verzeichnis "Resources", in dem Sie die Datei plazieren.

Windows 2000/XP/Vista

Installation als Benutzer

Kopieren Sie psi_de.qm nach %USERPROFILE%\psidata.

Systemweite Installation

Kopieren Sie psi_de.qm nach C:\Programme\Psi. (Wenn Sie Psi an einen anderen Ort installiert haben, passen Sie den Pfad entsprechend an.)
