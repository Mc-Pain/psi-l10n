اردو زبان پیک کے لیے تنصیب گائیڈ

1) اس صفحے پر "Psi translation - Lang" پر کلک کر کے زبان پیک ڈاؤن لوڈ کیجیے۔

2) مسل کو اپنیے عملی نظام کے حساب سے کسی مناسب ڈائریکٹری میں ڈالیں:

GNU/Linux ۔ psi_ur_pk.qm کو HOME/.psi/$ میں نقل کیجیے

Mac OS X ۔ psi_ur_pk.qm کو /Applications/Psi.app/Contents/Resources میں نقل کیجیے

ایسا کرنے کے لیے ایپلیکیشن پوشے میں Psi پر دایاں کلک کریں، پھر Show Package Contents پر کلک کریں۔ پہلے Contents پوشے میں جائیں، پھر Resources میں، اور یہاں اپنی psi_ur_pk.qm رکھ دیں۔



Windows 95/98/Me ۔ psi_ur_pk.qm کو C:\Program Files\Psi\psidata میں نقل کیجیے (اگر آپ نے Psi کہیں اور نتصیب کیا ہے تو راہ تبدیل کر لیجیے)۔

Windows 2000/XP ۔ اس کے دو طریقے ہیں:

ا) psi_ur_pk.qm کو C:\Program Files\Psi میں نقل کیجیے (اگر آپ نے Psi کہیں اور نتصیب کیا ہے تو راہ تبدیل کر لیجیے)۔
ب) اگر آپ کو کمپیوٹر پر منتظم رسائی نہیں ہے تو psi_ur_pk.qm کو C:\Documents and Settings\USERNAME\psidata میں نقل کر لیجیے (اور یہاں پر اپنا اصل نام صارف ڈالیں)۔


Windows Vista ۔ اس کے بھی دو طریقے ہیں:

ا) psi_ur_pk.qm کو C:\Program Files\Psi میں نقل کیجیے (اگر آپ نے Psi کہیں اور نتصیب کیا ہے تو راہ تبدیل کر لیجیے)۔
ب) اگر آپ کو کمپیوٹر پر منتظم رسائی نہیں ہے تو psi_ur_pk.qm کو <systemdrive>:\<localized string for "users">\<username>\psidata میں نقل کر لیجیے (اور یہاں پر اپنا اصل نام صارف ڈالیں)۔
زبان خود بخود سیٹ سیٹ ہو جاتی ہے کیونکہ Psi آپ کی زبان کے لیے آپ کا نظام متغیرہ ڈھونڈتا ہے۔ اگر آپ دستی طور پر Psi کو اپنی زبان پر بدلنا چاہتے ہیں (یا اگر یہ آپ کو کرنا پڑے) تو یہ آپ Profiles میں کر سکتے ہیں۔
