Guida all'installazione per il pacchetto della lingua italiana

Note Generali

Scarica il pacchetto della lingua cliccando sul link "Psi translation - Italiano" all'inizio di questa pagina.
Posiziona il file nella directory appropriata per il tuo sistema operativo come descritto nelle istruzioni seguenti.
La lingua e` impostata automaticamente usando le impostazioni del tuo sistema. Se vuoi cambiare lingua in Psi manualmente puoi farlo in "Profili".

GNU/Linux

Installazione come Utente

Copia psi_it.qm in $HOME/.config/psi/

Mac OS X

Installazione Sull'intero sistema

Copia psi_it.qm in Applications/Psi.app/Contents/Resources/
Per farlo clicca on il destro su Psi nella cartella dell'applicazione, quindi clicca su "Show Package Contents". Vai nella cartella "Contents" e poi "Risorse". Ora posizionaci il tuo psi_it.qm.

Windows

Installazione come Utente

Copia psi_it.qm in %APPDATA%\Psi.

Installazione sull'intero sistema

Copia psi_it.qm in C:\Program Files\Psi. (Se hai installato Psi da qualche altra parte, assicurati di cambiare l'indirizzo.)
